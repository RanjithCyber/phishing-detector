# Phishing Detection Tool 🔐

This is a cybersecurity tool that detects phishing websites using rule-based analysis.

## Features
- URL validation
- Domain age detection (WHOIS)
- Redirect chain analysis
- Brand impersonation detection
- Homograph attack detection
- Webpage login form analysis
- Risk score generation (0–100)

## Tech Stack
- Python
- Requests
- BeautifulSoup
- Whois

## How It Works
The tool analyzes a given URL using multiple phishing indicators and calculates a risk score to determine whether the website is safe or malicious.

## Note
This tool uses rule-based detection (not a trained AI/ML model).
